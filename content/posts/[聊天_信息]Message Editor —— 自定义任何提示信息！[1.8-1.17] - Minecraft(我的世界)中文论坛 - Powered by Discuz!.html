Message Editor 2.4.9
By jaq0bb

便捷的自定义你服务器中的信息吧！
NOW UPDATE 1.17! 2.4.9 BOSSBAR FIX!


前言
There are a lot of messages that are either unmodifiable or are modifiable but only by installing additional plugins. Why would you do that? Now you can install Message Editor and actually edit both without installing more plugins.

有很多消息是不能修改的或者只能通过安装其他拓展插件来修改。现在你可以安装 Message Editor 来进行修改而无需安装其他插件。


功能

With Message Editor you can edit existing messages that require additional plugins (like join message). However, Message Editor also offers editing messages that are unmodifiable by default and requires you to for example override the whole feature (like /version command).
You will find Message Editor very useful if there are messages you simply cannot edit but want to better customize your server.

使用 Message Editor，你可以编辑需要其他插件才可以修改的提示消息（例如 加入消息 JoinMessage）。Message Editor 还提供了修改原版目前无法被修改的提示消息的功能（例如 /version 指令的提示）。
如果你发现你想修改一些在安装 Message Editor 之前无法被修改的消息来更好地自定义你的服务器，你会发现 Message Editor 非常有用。

可修改的提示消息列表：
+ 聊天内容 - v1.0.0
+ Action Bar 提示内容 - v1.0.0
+ 踢出/断开连接消息 - v1.1.0
+ 取消发送消息 - v1.2.4
+ Boss Bar 消息 - v1.3.0 (需要服务器版本 至少 1.9)
+ 仅修改特定位置的消息 - v1.3.2
+ 更改提示消息位置 (例如 聊天栏 --> Action Bar) - v1.4.0
+ 修改计分板标题 - v1.5.0
+ 修改计分板内容 - v1.6.0
+ 使用 GUI 修改提示消息 - v2.0.0+ 修改容器标题 - v2.2.0
+ 修改物品名称 - v2.3.0
+ 修改物品描述 - v2.3.0
+ 修改实体名称 - v2.4.0

命令 & 权限 & 配置文件

命令
/messageeditor - 插件主命令 获取帮助
/message-editor - 同理 附属参数相同 以下使用第一个主命令作演示

/messageeditor reload - 重载插件
/messageeditor edit <message ID> - 修改消息提示 # 修改有变量的消息提示时会覆盖掉其中的变量，但可以手动再次添加。（感谢 @Light² 的提醒）
/messageeditor activate <message places> - 启用该消息提示位置
/messageeditor deactivate <message places> - 禁用该消息提示位置
/messageeditor deactivate-all - 禁用所有消息提示位置


权限

玩家无权限
OP 默认所有权限


配置文件
update:
  notify: true
# 是否启用更新提示

attach-special-hover-and-click-events: true
# 是否启用悬停点击来修改提示信息

message-edits:
# 为修改的提示信息记录，暂不做举例
