A Minecraft Java Release Candidate
一个 Minecraft Java 版的发布候选版


We're now releasing the first (and hopefully only) release candidate for Caves & Cliffs: Part I. If there are no major issues following this release, no further changes will be done before the full release.
我们现在发布 1.17 （洞穴与山崖第一部分）的第一个发布候选版（希望是唯一一个）。如果我们在该版本中没有发现重大问题，在6月8日发布正式版本之前我们不会再做改动。

Happy mining!
挖矿愉快！

  • MC-227323 - 自定义玩家头颅放置到盔甲架上后可能会闪

  • 优化了在高延迟状况下离开船时的失去同步问题
  • Fixed crashes
  • 修复崩溃

Get the Release Candidate
获取此发布候选版

Release candidates are available for Minecraft: Java Edition. To install the release candidate, open up the Minecraft Launcher and enable snapshots in the "Installations" tab.
该发布候选版在 Minecraft Java 版中可用。若要游玩该版本，打开启动器的“配置”标签，并启用快照版本即可。

Release candidates can corrupt your world, so please make a backup and/or run them in a different folder from your main worlds.
发布候选版可能导致你的存档损坏，所以请在游玩前做好存档备份/或将该版本运行于另一个文件夹中。
