每周快照是Minecraft的测试机制，主要用于下一个正式版的特性预览。
然而，每周快照主要用于新特性展示，通常存在大量漏洞。因此对于普通玩家建议仅做测试尝鲜用。
适用于正式版的MOD不兼容预览版，而且大多数MOD都不提供对每周快照的支持。
Minecraft 1.7（世界生成器更新）仍未发布，13w38c为其第7个预览版。
注意！旧版启动器已经无法启动Minecraft 1.6以及1.7快照，请看下面的说明。
本版新闻除非特别说明，否则均允许转载，但转载必须注明本帖地址。

13w38b&c修正38a里面的图像漏洞、崩溃，还有物品展示框的问题

官方新版（正版）启动器下载地址：
Windows: https://s3.amazonaws.com/Minecra ... ncher/Minecraft.exe
Mac OS: https://s3.amazonaws.com/Minecra ... ncher/Minecraft.dmg
全平台: https://s3.amazonaws.com/Minecra ... ncher/Minecraft.jar

自Minecraft 1.7快照开始，Minecraft中文论坛不再提供快照客户端的下载。请各位玩家使用启动器自行下载体验。
对于正版用户，可以直接接收到新版官方启动器的快照推送。前提是：
  • Profile中的Version Selection勾选Enable experimental development versions ("snapshots")一项
  • Use version选择“Snapshot 版本号”或“Use latest version”
对于非正版用户，前提是：
  • 你使用的启动器能够在线读取官方的版本列表，并具备下载功能。具体细节请询问你的启动器作者。

服务端：
  • jar: https://s3.amazonaws.com/Minecraft.Download/versions/13w38c/minecraft_server.13w38c.jar
  • Windows EXE: https://s3.amazonaws.com/Minecraft.Download/versions/13w38c/minecraft_server.13w38c.exe
