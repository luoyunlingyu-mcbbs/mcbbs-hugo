插件介绍
SafariNet 是一个生物捕捉插件，可让您的玩家捕捉和释放每一个生物（可自定义捕捉/释放的生物）。这对运送动物起到了极大的帮助，比如运送村民，这会方便许多，而且支持使用发射器生成某个生物。这个插件有什么特别之处？此插件适用于NBT。这意味着这个插件存储了生物的所有信息，比如马鞍、马匹变体、驴箱、自定义名称、生命值、村民职业（交易物品信息）和盔甲。
官方WIKI（英文）：SafariNet Spigot Plugin | SpaceIO Plugin Documentation

特征
  • 兼容1.8 到 1.20.x
  • 支持模组包生物
  • 保存整个实体数据，包括村民具体交易内容，马和骡子上的箱子，健康状况（血量），羊的颜色等。
  • 拥有一次性和可重复使用的捕捉蛋
  • 可自定义的 SafariNet 项目（支持模型数据）
  • 生物通过发射器生成或单击右键
  • 可配置的合成方法
  • 用于制作刷怪蛋的权限节点（可重复使用和一次性使用刷怪蛋）
  • 使用 SafariNet 更改生成器类型
  • VIP玩家捕捉某些生物
  • 可设置玩家可捕捉某些生物
  • 为某些生物设置捕获几率
  • 捕捉蛋高度自定义（可设置每个实体类型应显示的内容）

命令和权限

命令 所需权限 作用
/safarinet 可简写为 /sn / 主命令
/safarinet reload safarinet.reload 重新加载配置文件（admin）
/safarinet give <player> <singleuse/reusable> [amount] safarinet.give 给予<player><一次性/无限次的捕捉蛋> [数量]（admin）
/safarinet set <singleuse/reusable> safarinet.set 将手中的捕捉蛋设置为一次性或无限次（admin）
/safarinet recipe <singleuse/reusable> safarinet.recipe 将配方增加到聊天中        （user）
/safarinet inspect safarinet.inspect 显示存储在手中捕捉蛋中的实体的 NBT 字符串(admin)
权限 作用
safarinet.craft.singleuse 允许制作一次性捕捉蛋
safarinet.craft.reusable 允许制作无限次捕捉蛋
safarinet.use 用于释放和捕获生物
safarinet.use.singleuse 只允许使用一次性的捕捉蛋捕获
safarinet.use.reusable 只允许使用无限次的捕捉蛋捕获
safarinet.release.<mob> 允许释放此生物类型，必须在配置中启用。
safarinet.capture.<mob> 允许捕获此生物类型，必须在配置中启用。
safarinet.releasemob.use, safarinet.capturemob.use 这两个权限可以用来代替safarinet.use来允许每个人释放生物，同时只允许VIP等级（或其他）捕获它们
safarinet.changespawnertype.primitiv.<entity-type> 允许更改更多生成器类型
