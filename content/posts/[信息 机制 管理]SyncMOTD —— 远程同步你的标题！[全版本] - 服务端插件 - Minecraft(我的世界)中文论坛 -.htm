SyncMOTD

想要同步其他服务器的MOTD或者同步本地的多个BC的MOTD
使用它，你将会发现一个新大陆！
盗版服狂喜(doge)

如果你有多个BC服需要设置多个MOTD
在Redis中，只设置一个服务器的MOTD就可以实现，而不是在所有BC服务器上设置MOTD

插件的一般用法可以同步像MOTD、PlayerList等数值
还可自定义开启同步目标服务器的IP、端口、图标设置

插件使用示例

汉化后的配置文件
config.yml (675 Bytes, 下载次数: 42)
2021-2-7 18:17 上传
点击文件名下载附件
仅供参考，格式以上方附件为准！
  1. motd:
  2.   useFavicon: true  # 多线程同步服务器图标
  3.   useMaxPlayers: true # 多线程同步最大人数
  4.   useMotd: true  # 多线程同步服务器MOTD
  5.   useSample: true # 多线程同步样品服务器
  6.   remote:
  7.     use: true  # 是否开启同步
  8.     updateTime: 5  # 同步间隔
  9.     server:  # 远程同步的服务器信息
  10.       IP: 'mc.hypixel.net'  # 服务器IP
  11.       Port: 25565  # 服务器端口
  12.     proxy:  # 代理设置
  13.       use: false  # 代理开关(true是开启，反之则关闭)
  14.       server:
  15.         IP: '188.166.34.137'  # 代理IP地址
  16.         Port: 9000  # 代理端口
  17.         ProxyType: SOCKS  # 代理类型 (SOCKS或HTTP)
复制代码
注：修改config.yml配置文件后，请使用/syncmotd reload重载插件或重启服务器


该插件支持BungeeCord和Spigot
如外网可以使用代理链接，本质目的为了获取MOTD
不建议在IP栏中填写域名，请解析后使用

解析示例:123.123.123.123
