　▋　✨ 特色 Features

  • 多样任务：攻击铁傀儡，猎杀玩家，制造木桶，杀死凋零骷髅，还有更多任务等你探索！
  • 群组服兼容：既可以私服中自由游玩，手动开启游戏，加入游戏，也可以放在群组服中，游戏自动开启，自动加入游戏！
  • 配置简单：完整配置指引，配置结构明晰。
  • 高度自定义：文本、任务等均可自定义。
  • 拓展性强：开源，开放 API 供开发者自定义任务、自定义游戏模式等（GPLv3）。
  • 可能还有更多……？

　▋　🚀 安装与使用 Install & Usage

  • 注意：开始游戏后玩家背包、出生点等几乎一切东西都会清空，请不要直接安装在生存服上！建议另开一个服务器进行游玩。在未来的版本里，将会保存玩家背包等信息，请敬请期待！

  • 下载本插件（见上一节），将其复制进服务端的 plugins 文件夹中，重启服务器。
  • 默认情况下，有权限的玩家现在可以使用 /bingo start 命令来开启一局游戏，此时请耐心等待世界生成，这通常会在 30 秒内完成。
  • 世界生成完成后，其他玩家就可以通过 /bingo join 命令加入游戏，默认情况下有 4 人以上游戏才会开始（可设置）。
  • 玩的开心！

    指令与权限：
  • /bingo gui 打开GUI。 bingo.use.gui
  • /bingo gui <玩家> 打开他人的GUI。 bingo.use.gui.others
  • /bingo join 加入一场Bingo游戏。 bingo.use.join
  • /bingo leave 离开一场Bingo游戏。 bingo.use.leave
  • /bingo start 开始一场Bingo游戏。 bingo.admin.start
  • /bingo stop 停止一场Bingo游戏。 bingo.admin.stop
  • /bingo reload 重载Bingo游戏的配置文件。 bingo.admin.reload
  • /bingo config 改变一些设置。 bingo.admin.config


　▋　🔧 配置 Setup

  • config.yml:
  1. lang: en_US # 默认为英文， 支持 zh_CN（简体中文） 和 zh_TW（繁体中文）。

  3. room:
  4.   # 这是 Bingo 游戏的最多玩家数。
  5.   max-player: 8

  7.   # 这是 Bingo 游戏的最少玩家数。
  8.   min-player: 4

  10.   # 这是 Bingo 游戏胜利玩家的数量。（达到胜利玩家数后游戏会立刻结束）
  11.   # 建议值： 玩家上限 / 4
  12.   winner-count: 2

  14.   # 这是当一名玩家胜利后，距离游戏自动结束还有多少秒。
  15.   # 小于 1 禁用。
  16.   end-time: 300

  18.   # 这是开始游戏的时间。
  19.   # 例子:
  20.   # start-time:
  21.   #  - "4|300"  -  当玩家数大于等于4时， 游戏将在300秒内开始。
  22.   #  - "6|60"  -  当玩家数大于等于6时， 游戏将在60秒内开始。
  23.   #  - "8|5"   -  当玩家数大于等于8时， 游戏将在5秒内开始。
  24.   # *你需要写上最少玩家时的开始时间。
  25.   start-time:
  26.     - "4|300"
  27.     - "6|60"
  28.     - "8|5"

  30.   # 是否允许玩家在游戏中途进入。
  31.   join-while-game: true

  33.   # 这是Bingo游戏世界名称。请在开始前务必检查此项！！
  34.   world-name: bingo

  36.   # 这是游戏结束后，玩家将被传送至的世界。
  37.   main-world: world

  39. chat:
  40.   # 当一名玩家完成一项任务后，是否会在聊天栏中显示，
  41.   # 可用值：true / false
  42.   complete-task-show: true

  44.   # 当一名玩家胜利后，是否在聊天栏中显示。
  45.   # 可用值：true / false
  46.   complete-game-show: true

  48. game:
  49.   # PVP保护时间。
  50.   # 小于1禁用。
  51.   no-pvp: 300

  53.   # 地图边界的大小。
  54.   # 小于1禁用。
  55.   world-border: 3500

  57.   # 预生成的区块的范围。
  58.   # 小于1禁用。
  59.   pre-generate-range: 12

  61.   # 初始时随机传送的最远距离。
  62.   # 小于1禁用。
  63.   random-teleport-range: 800

  65.   # 是否允许玩家打开他人的GUI。 (bingo.use.gui.others)
  66.   # 可用值：true / false
  67.   access-to-others-gui: true

  69.   # 玩家的初始装备。
  70.   # 如果你不熟悉，你应该在游戏中来配置此项。
  71.   # 使用 "/bingo config startkits" 获得更多帮助。
  72.   startkits: []


  75. display:
  76.   # 改变任务列表的物品。
  77.   # 如果值为 FILLED_MAP （地图）， 那么地图中会显示你的任务。
  78.   item-list-material: FILLED_MAP

  80. server:
  81.   # 如果设置为 true， 那么当服务器开启时会自动开始游戏， 并且玩家进入时会自动加入游戏， 游戏结束后会自动重新开启。
  82.   # 可用值：true / false
  83.   auto-start-end: false

  85.   # 启用 Bungeecord 支持。
  86.   # 可用值：true / false
  87.   bungee: false

  89.   # 游戏结束后玩家会被传送至哪个服务器。
  90.   # 仅当 auto-start-end 和 bungee 都为 true 时生效。
  91.   lobby-server: lobby

  93. # 当游戏开始时是否重新生成世界， 仅用于测试。
  94. debug: false
