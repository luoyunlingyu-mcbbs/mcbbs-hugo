顾名思义，就是本作的介绍 Orz
大家好，我是猫狗喵
今天要带来的是大(老)家(人)的童年，枫之谷的还原地图
可能有一些人已经看过制作纪录系列了，不过还是简单介绍一下 MapleCraft 这个作品吧
(第一次发这种还原类的作品，如果有违反版规麻烦再通知我，我会尽速删除)

MapleCraft 顾名思义是 MapleStory 与 Minecraft 的结合，目标是在 Minecraft 中尽可能还原 BB 大改前的枫之谷所有内容。不过碍于现实因素，说不准哪时候就会弃坑(社畜生活虎视眈眈)
本来只是入伍前想说闲著协助一下 テリ 實現他的夢想，没想到替代役生活意外挺闲(?) 所以就这样做了半年。不过枫之谷这么大一款游戏，当然不可能半年就完整的还原出来，所以现在就来简单介绍一下目前做了什么吧！
1. 从初心者到二转四大职业所有主动与被动技能
2. 能力点数自由配置
3. 几乎所有部位的装备功能
4. 药水与卷轴功能
5. 3D 化怪物
6. 完整的超级绿水灵组队任务流程
从伤害、命中率，到动作延迟、DEBUFF 叠加效果等等，几乎所有效果都经过严谨考究。基本上你想的到想不到的细节都还原进去了，如果有发现没还原的部分，应该几乎都是我觉得还原的成本效益太低所以才没做进去

·/ 补充说明 /·
一些说明的补充
这张地图的主旨在于勾起各位老人们的童年回忆，玩家可以在村庄自由切换职业、快速升级等等以便体验各个职业的技能，并且与朋友一起回忆进行超级绿水灵组队任务的点点滴滴
注意：游玩这张地图需要部分旧枫之谷（冒险岛）常识，建议完全没接触过旧枫之谷（冒险岛）的玩家找有玩过的玩家陪同

只要有人选择开始组队任务，所有在村庄内 21 等以上的玩家都会一同参与。另外这个版本的超级绿水灵没有队长的设计，任何玩家都可以与 NPC 对话尝试完成关卡。而为了让玩家体验更丰富的内容，组队任务最后的奖励也增加了很多种类的装备、卷轴、飞镖等等

虽然是希望尽量还原枫之谷的内容，不过有些东西还是有针对 Minecraft 与现代人的游戏步调去做修改。例如超绿的入场等级从 21~30 改成 21 以上即可，也取消了 4 个人的人数限制(边缘人友善设计)

目前地图支援繁体中文与英文(US)两种语言，如果有人有兴趣帮忙做韩文或日文之类的翻译并推广到该区社群的话欢迎私信联络～
另外如果对这张地图的技术有兴趣的可以左转制作纪录的文章，有很多应用技术的解析。想追踪最新消息也可以关注我的 YT 频道
TIP：需要特殊方法才能访问 嘶 你们懂得

TIP：搬运君已经弄完繁转简，但由于当初并未玩过冒险岛（只玩过冒险岛2），物品或技能名称可能有误，欢迎提出错误，搬运君会采纳修正。待修正完全，会上传给原作者那。关于部分GUI未汉化，因为我自己弄还是挺麻烦了点，只把装备介绍的标签给汉化了下（当然可能有错误），如有需要，可以说一下还需要让我汉化GUI的地方，或者有擅长像素图制作的，可以私信我（为爱发电，并不接受有偿 Orz）
TIP：语言、物品、GUI都在材质包里，可以在那里改，改好直接发我也行www

·/ 注意事项 /·
游玩前需要注意的地方
1. 请先安装完材质包，依照绿字提示完成设定后再进入地图
2. 使用第三人称战斗可以有更好的视觉体验
3. 善用 F3+D 清除聊天栏，可以有效避免视线遮蔽问题
4. 议不要使用游戏内置的聊天视窗打字，以免干扰他人操作 UI 界面。要避免受到他人干扰可以将「聊天设定」中的「聊天栏」设为「只显示指令」
5. 使用 /kill 或 /tp 指令有极高概率导致 BUG
6. 使用服务器游玩可能因快速发送信息而被踢出，使用局域网可以避免此问题
7. 我并未拥有任何枫之谷（冒险岛）图片或音乐/音效的所有权，请勿自行散布素材档案

另外本地图有部分设定调整功能，可以参考下面指令进行设定(需要OP权限)
· 调整经验值倍率：/scoreboard players set #WORLD_EXP_RATE const <倍率>
· 调整道具掉落率：/scoreboard players set #WORLD_DROP_RATE const <倍率>
· 调整枫币掉落量：/scoreboard players set #WORLD_MESO_RATE const <倍率>
· 困难模式(怪物等级、血量、攻击力等大幅增加)：/scoreboard players set #HARDCORE const 1
· 关闭困难模式：/scoreboard players reset #HARDCORE const
