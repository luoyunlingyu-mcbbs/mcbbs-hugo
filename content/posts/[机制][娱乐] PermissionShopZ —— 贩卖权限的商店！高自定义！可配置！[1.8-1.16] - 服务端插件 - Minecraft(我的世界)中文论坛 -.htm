PermissionShopZ
权限商店！超高自定义的配置！
介绍-Introduction
       PermissionShopZ是一款可以让玩家购买权限的插件！可以进行超高自定义配置的一款插件且配置过程极为简单！啥？听说你服务器的玩家想要更多的权限？这款插件可以完美的满足你！玩家可以使用游戏币购买权限！
简简单单的效果图
效果图(图示为随身工作台权限)


注意事项-Notice
1.该插件必须需要TabuuCore否则将无法正常运行。TabuuCore可以通过SpigotMC或GitHub下载。
2.此插件使用Github开源，请遵守开源协议|该插件的Github|
3.在使用插件时，若出现任何BUG，请务必前往作者的Github下提交问题
4.此插件所支持的权限管理器：
  • GroupManager
  • PermissionsEx
  • LuckPerms
  • Vault
所有其他权限管理器都必须在config.yml中进行配置


插件功能-Work
该插件可以让玩家购买权限


插件指令-Command
      /pshop ························································································打开权限商店的GUI

      /pshop add <名称> <价格> <权限> <更多权限……>············向权限商店中添加权限

      （在商店GUI中所显示的物品外观是你输入此指令是手中所拿的物品），格式一定要正确！否则指令将无效！

      /pshop remove··········································································打开删除权限商店的GUI

      /pshop reload ···········································································重载配置


权限节点-Permissions
  1. permissionshop.default : 允许玩家打开商店GUI
  2. permissionshop.admin  ：允许玩家管理商店

  15. # 默认设置为："VAULT" （请填入你服务器所装载的权限管理系统）
  16. PermissionManager: "VAULT"

  18. # DESCRIPTION:  The command executed when someone buys a permission.
  19. # FORMAT:       String
  20. # NOTE:         Only use this when PermissionManager is set to 'CUSTOM'.
  21. # 描述： 有人购买权限时执行的命令
  22. # 格式： 字符串 （String）
  23. # 注意： 仅当PermissionManager设置为'CUSTOM'时才使用此选项
  24. CustomPermissionCommand: "manuaddp {PLAYER} {PERMISSION}"

  26. # DESCRIPTION:  GUI Size
  27. # FORMAT:       InventorySize (http://u.tabuu.nl/5mh7vwb2x).
  28. # NOTE:         The inventory needs at least 3 rows.
  29. # 描述： GUI的尺寸大小
  30. # 格式： InventorySize（GUI大小后方网站上有对应的bukkit名称）（http://u.tabuu.nl/5mh7vwb2x）
  31. # 注意： GUI的大小至少要有3行
  32. GUISize: "THREE_ROWS"   #"THREE_ROWS"表示GUI有三行

  34. # DESCRIPTION:  Material of the unlocked items.
  35. # FORMAT:       XMaterial (http://u.tabuu.nl/5mh5duwn0).
  36. # NOTE:         If you do not want it to change leave it empty.
  37. # 描述： 在商店GUI中对于已经购买的权限后，GUI中所显示的“物品”
  38. # 格式： XMaterial | 材料的bukkit名称在网站中（http://u.tabuu.nl/5mh5duwn0）
  39. # 注意： 如果你不希望进行更改，那请将"UnlockedMaterial"设置为空
  40. UnlockedMaterial: "IRON_BARS"  #"IRON_BARS" 铁栅栏

  42. # DESCRIPTION:  Whether or not the perk should display the permissions it contains.
  43. # FORMAT:       Boolean
  44. # 描述： 商店GUI中是否会显示所购买的具体权限（例如："essentials.enderchest"这种）
  45. # 设置为ture时，意味着玩家可以看到所购买的权限的具体“值”
  46. # 设置为flase时，意味着玩家无法观看所购买的权限具体的“值”
  47. # 格式： Boolean
  48. DisplayPermissionList: true

  50. # DESCRIPTION:  Whether or not the messages should be displayed as titles instead of in chat.
  51. # FORMAT:       Boolean
  52. # 描述： 消息是否应显示为标题而不是在聊天框中
  53. # 格式： Boolean
  54. # 设置为true时，意味着信息将会以“标题”的形式显示在你物品栏的上方
  55. # 设置为false时，意味着信息将会显示在你的聊天框中
  56. UseTitle: true

  58. # DESCRIPTION:  Whether or not zeros in the price should be replaced with their corresponding suffix.
  59. # FORMAT:       Boolean
  60. # NOTE:         The suffixes can be configured below.
  61. # 描述： 价格中的零是否应替换为其相应的后缀，比如3000会变成3K；4500000会变成4.5m
  62. # 格式： Boolean
  63. # 注意： 您可以在下面配置后缀。
  64. # 设置为true时，意味着开启数字后缀，即3000会变成3K；4500000会变成4.5M等，这个后缀是可以自定义配置的
  65. # 设置为flase时，意味着关闭数字猴嘴，即数字不会缩减拥有后缀。3000所显示的就是3000；4500000显示的就是4500000
  66. UseNumberSuffix: false

  68. # DESCRIPTION:  Suffixes per zero count.
  69. # FORMAT:       'zero count': 'suffix'
  70. # 描述： 每个零计数的后缀。多少个零所对应的后缀
  71. # 格式： '数字': '后缀'
  72. NumberSuffixes: #数字后缀
  73.   '3': 'k'  #3个0会将该数字的后缀变成K；例如3000会变成3K
  74.   '6': 'M'  #6个0会将该数字的后缀变成M；例如4500000会变成4.5M
  75.   '9': 'B'  #9个0会将该数字的后座变成B；例如9000000000会变成9B
