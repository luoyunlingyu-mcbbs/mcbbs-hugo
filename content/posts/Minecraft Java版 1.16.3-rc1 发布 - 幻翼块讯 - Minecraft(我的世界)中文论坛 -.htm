Minecraft Java版是指Windows、Mac OS与Linux平台上，使用Java语言开发的Minecraft版本。

候选版已可供普通玩家进行抢鲜体验，但仍需当心可能存在的漏洞。

Minecraft 1.16.3 仍未发布，RC1 为其第 1 个预览版。

转载本贴时须要注明原作者以及本帖地址。



NEWS
Minecraft 1.16.3 Release Candidate 1
Minecraft 1.16.3 发布候选 1

A Minecraft Java Release Candidate
Minecraft Java 发布候选

What better way to start the week than with a new release candidate that fixes two bugs? 1.16.3 addresses a duplication exploit, along with a pathfinding issue.
还有比发布修复了2个漏洞的候选版本更好的办法去开始新的一周吗？1.16.3修复了复制物品和寻路的漏洞。

Fixed bugs in 1.16.3 Release Candidate 1

修复于 1.16.3 发布候选 1 的漏洞

  • MC-198678 - Giving an item and a gold ingot to a baby piglin and killing it duplicates the item
  • MC-198678 - 给予幼年猪灵一个物品和一个金锭然后杀死它会得到双倍的原物
  • MC-196449 - Piglins, Piglin Brutes, Hoglins and Zoglins have trouble pathfinding to the player when attacking
  • MC-196449 - 猪灵、猪灵蛮兵、疣猪兽和僵尸疣猪兽在攻击玩家时出现寻路漏洞

Get the release candidate
获取发布候选

Release candidates are available for Minecraft: Java Edition. To install the release candidate, open up the Minecraft Launcher and enable snapshots in the 'Installations' tab.
发布候选在Minecraft Java版中可用。想要安装发布候选，打开 Minecraft 启动器 并在“配置”选项卡里启用快照。

Release candidates can corrupt your world, so please backup and/or run them in a different folder from your main worlds.
发布候选可能会损坏你的存档，所以请备份或者在不同于你主要游玩的存档所在的文件夹里运行。
