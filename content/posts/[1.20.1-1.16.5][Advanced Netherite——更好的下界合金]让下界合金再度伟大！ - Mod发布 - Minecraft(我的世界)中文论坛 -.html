更好的下界合金

i  前言
注意！该模组具有 Forge 和 Fabric 两个版本，其中 Fabric 版本需要 Fabric API 作为前置！
本模组 Forge 和 Fabric 版本差别较大，具体差异可见下文疑难解答中的“版本差别”一栏。模组作者的开发重心更偏向于 Forge 版，因此也更推荐 Forge 版

在原版 MC ,下界合金已经算是最顶级的材料了。但如果能用原版的其他材料做出比一般的下界合金更强大的材料会怎样呢？这个衷于原版的模组便为此给出了答案：本模组加入了4种全新的下界合金，均可用原版的资源制成。这4种下界合金也可以制作出多种实用的装备和盔甲，大大丰富了玩家们的原版生存游戏体验。对于那些热衷于游玩原版纯净生存的玩家来说，想要更轻松的体验到更好的武器盔甲，这个模组无疑是一个不错的选择


i  模组介绍
开始的最基本介绍。。。


本模组加入了4种全新的下界合金以及对应的装备和盔甲。每种全新的下界合金都可由（原版）下界合金和原版其他特定矿物合成而来。这4种下界合金也可以打造出更好的全新的工具，武器和盔甲，它们不仅有着更高的效果数值，甚至也具有额外的附加功能。



这4种下界合金的等级从低到高分别是：镶铁的下界合金，镶金的下界合金，镶绿宝石的下界合金，镶钻石的下界合金。每种下界合金均可由上一等级的下界合金通过与对应等级相同的材料结合来升级。具体升级流程可见下图：

  
  

之后，你就可以使用新的下界合金对各种原版下界合金的道具，武器和盔甲进行升级强化了。这里以剑为例，不同等级的下界合金剑的升级流程如下所示：

  
  

本模组加入的其他道具，武器和盔甲的升级流程也与之类似，这里就不作过多解释了。你可以访问模组的官方 Wiki 以查阅各物品的配方，或者安装 JEI 以随时随地查阅配方！

那么，用了全新的下界合金制成的装备有什么特点呢？



对于这些全新的下界合金系列装备，每种都有着其独特的附加效果（工具类的额外掉落和盔甲类的震慑效果等）。对于修复，只需使用与之对应的下界合金在铁砧处进行修复即可。以下为各种全新下界合金装备的特点：

1. 更高效率 + 额外掉落（镐 / 锄 / 斧 / 铲）
使用了更好的下界合金制成的各类道具，相较于（原版）下界合金的而言，采集速率更快，耐久值更高（且不同类型的下界合金的等级越高，这些数值也越高）。此外，对于镐而言，挖掘矿石的时候还会有额外的矿石掉落加成（言外之意是，不用附上时运，挖掘单个矿石能也能获得更多的产物）。具体加成情况可见官方 Wiki 对此的说明

2. 更高伤害 + 额外掉落（剑）
使用了更好的下界合金制成的剑，相较于（原版）下界合金的而言，攻击伤害更高，耐久值也更高（且不同类型的下界合金的等级越高，这些数值也越高）。此外，击杀生物的时候还会有额外的掉落物加成（言外之意是，不用附上抢夺，击杀单个生物能也能获得更多的产物）。具体加成情况可见
官方 Wiki 对此的说明

3. 更强保护 + 特别震慑效果（盔甲）



使用了更好的下界合金制成的盔甲，相较于（原版）下界合金的而言，保护效果更好，耐久值更高（且不同类型的下界合金的等级越高，这些数值也越高）。此外，不同类型的下界合金盔甲还有着对特定生物的震慑功能，即穿上特定的盔甲，就能让某些原本对玩家敌对（或条件敌对）的生物不敢主动攻击玩家。其中：

镶铁的盔甲对幻翼有着震慑作用；
镶金的盔甲对猪灵有着震慑作用；
镶绿宝石的盔甲对末影人有着震慑作用；
镶钻石的盔甲对猪灵和末影人都有着震慑作用！

此外，你还可以自定义各种盔甲的震慑属性等参数（需要 Configured 模组的支持）。对于各种盔甲的其他参数信息，可见官方 Wiki 对此的说明

还想要了解更多吗？访问官方 Wiki（英文）来找到你想要的！

Q：该模组能否向更低的 MC 版本（如1.12.2）移植？
A：抱歉，不能。因为（原版）下界合金在 1.16 被加入，所以该模组最低也仅支持 1.16

Q：如果我遇到了问题，我应该去哪里反馈？
A：您可以在本模组的官方 Discord 频道进行反馈。如果您遇到技术上的问题建议前往官方 GitHub 进行反馈

Q：啊？！为什么镶钻石的下界合金比镶绿宝石的下界合金还要好？
A：
Diamond ore may be more common than Emerald ore, but there are other ways to create infinite sources of emeralds making them more common.
Think about: trading, raids, looting chests, etc.
No, I'm not going to change this because you don't like doing any of the above.


如果仅从矿石考虑，钻石确实比绿宝石常见，但通过一些方法可以获得几乎无限多的绿宝石（如交易，袭击塔等）。所以这是作者有意为之的设定（确信）
