插件介绍
这是一个集记分板+TAB+称号+聊天于一体的插件.您可以在Config中设置所有内容！
此插件支持PermissionEx作为前置，并支持Vault的变量。
您还可以设置聊天的前缀(称号)。
它几乎与占位符(Papi)的用法相同，只不过您可以使用超过16个字符.
您还可以停用插件的各个功能，例如，如果您只想使用TAB列表或只想使用记分板。
您都可以在Config内设置。
常用变量
%playeronline%, %playermax%, %rang%, %geld%, %name%, %uhrzeit%, %welt% %ping%
当然你想要更多的变量.请联系作者来添加.（admin@xitecraft.de）作者是一个德国人.

配置文件
# 您可以在此处设置记分板、表格、前缀和聊天格式。
#
# 计分板设置:
# 空行时请这样写： - ' '
# 确保在每个空行后多留出一个距离！！
#  - ' '
#  - '  '
# 您可以使用以下变量： %playeronline%, %playermax%, %Rang%, %geld%, %name%, %uhrzeit%, %welt%
#
# TPB列表设置:
# 你可以使用以下变量: %playeronline%, %playermax%, %Rang%, %geld%, %name%, %uhrzeit%, %welt%, %newline%
#
# 前缀(称号)设置:
# 支持全部颜色.如&0-9.称号与权限组直接要有三个%%%分开(例如: '&7[&cOwner&7]%%%permission.owner' )
# 如果您激活了pexsupport，您将使用PermissionEx作为前置. 但是，您必须将权限组的名称放在%后面。
# 使用PermissionEX为前置.例子是这样写: '&7[&cOwner&7]%%%owner'
#计分板设置
scoreboard:
#是否开启计分板
  aktivieren: 'true'
#计分板的昵称
  name: Scoreboard
#计分板的内容
  scores:
  - '&cSpieler: '
  - '&6%playeronline% / %playermax%'
  - ' '
  - '&cDein Name:'
  - '&7%name%'
  - '  '
  - '&bDeine Welt:'
  - '&a%welt%'
  - 'Geld: %geld%'
  - 'Rang: %rang%'
  - 'Zeit: %uhrzeit%'
#TAB列表设置
tablist:
#是否开启TAB列表
  aktivieren: 'true'
#TAB列表的内容(上方)
  zeile1: '&eDie Tablist kannst du in der Config einstellen %newline% Zeile 1 (?ber
    den Spielern)'
#TAB列表的内容(下方)
  zeile2: '&6Tablist Zeile 2 (Unter den Spielern)'
#前缀设置
prefix:
#是否开启前缀设置
  pexsupport: 'false'
#设置前缀的称号与权限组
  list:
  - '&7[&cOwner&7] &7%%%permission.owner'
  - '&7[&bSup&7] &7%%%permission.owner'
  - '&7[&aSpieler&7] %%%permission.spieler'
