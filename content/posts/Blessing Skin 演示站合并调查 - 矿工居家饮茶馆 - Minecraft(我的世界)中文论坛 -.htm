MCBBS 的各位好，我们是 LittleSkin 的运营方 Honoka Tech LTD。

想必各位应该对 Blessing Skin 这个皮肤站程序有所耳闻吧？它的出现打破了 Skin Me 在国内皮肤站领域的垄断，同时让许多服务器甚至是个人都拥有了属于自己的皮肤站，原作者 printempw 维护的演示站点更是直接造福了超过三万用户（尽管这只是一个演示站点）。

随着 Blessing Skin Server v4 "Tamias" 的发布，Blessing Skin Server 的开发工作已由 Blessing Skin Community 接手，而 printempw 渐渐退出了主要的开发工作。遗憾的是，Blessing Skin 演示站仍然停留在 Blessing Skin Server v3.5.0，printempw 表示已经没有精力维护新版本的演示站了，甚至 Blessing Skin Server 在 MCBBS 的发布帖也停留在 v3.5.0（v4 版本的发布帖已由当前的主要维护者 @gplane 发布）。所以，我们有了这样的计划：将 Blessing Skin 演示站与 LittleSkin 合并。不过在实施合并计划前，我们想要先征求演示站用户的意见。

请注意：本次合并不是收购，这是我们友好讨论后的计划，不涉及任何金钱交易。这份调查对于未来演示站和 LittleSkin 的发展具有重大意义，请认真参与，谨慎投票。合并后 Blessing Skin Server 将由 Blessing Skin Community 继续独立开发，但演示站的所有数据将会被转移到 LittleSkin。我们不会强行合并两个站点，若调查结果显示支持率不足，我们将维持两个站点的现状，什么都不做。

各位可能对 LittleSkin 不甚了解。LittleSkin 建站于 2016 年三月，至今已有超过三年历史，并自 CustomSkinLoader 14.7 起被加入默认加载列表，加载次序仅次于正版皮肤。我们或许不是最专业的，但是我们一定是国内使用 Blessing Skin Server 部署的皮肤站里最用心的。我们从建站起即采用公益运营模式，基础服务不收取任何费用，也不会提供积分充值等增值服务。截至发帖时间，LittleSkin 的用户总数约两万四千人，高峰期每日处理超过 300 万次查询请求（当前单日最高处理超过 562 万次查询请求，这两个数据均来源于腾讯云 CDN 的统计数据），大概是国内除了 Skin Me 和 Blessing Skin 演示站外最大的 Minecraft 皮肤站。LittleSkin 使用 Blessing Skin Server 开发版本，目前处于活跃维护中，各种新特性和 Bug 修复都会在第一时间更新，并且拥有在中国大陆境内相较 Blessing Skin 演示站更快的访问速度以及实时刷新的角色信息缓存。值得一提的是，大多数 Blessing Skin Community 成员都是 LittleSkin 的站点管理员，几乎可以说 LittleSkin 就是由 Blessing Skin Community 维护的。需要注意的是，LittleSkin 不允许上传包含敏感内容的公开材质（即必须将敏感材质设为私密材质，敏感材质包括但不限于瑟琴敏感和政治敏感等），一旦发现有公开的敏感材质，这份材质会被删除，并且上传者将会被 LittleSkin 永久封禁。

如果最后决定合并两个站点，在合并后，演示站的数据将会被迁移至 LittleSkin，演示站的皮肤托管服务和 Yggdrasil 鉴权验证服务也将交由 LittleSkin 运营：
  • 大多数演示站用户的账号数据（包括积分、角色以及 Yggdrasil API 的数据）将会被无缝迁移至 LittleSkin
  • 如果两个站点出现同名角色，则演示站的角色数据将会被丢弃。
  • 如果两个站点出现邮箱相同的用户，我们会视为是同一个用户在两个站点创建的不同账号，这个用户除上传的皮肤外的在演示站的所有数据都将会被丢弃。
  • 如果演示站的皮肤库中出现了敏感材质，这些材质将会在鉴定后被自动设为私密材质。
  • 由于 LittleSkin 已被加入 CustomSkinLoader 的默认加载列表，只要您使用的皮肤 Mod 不是远古版本的 CustomSkinLoader 或其它皮肤 Mod，您无需更换皮肤 Mod 或修改配置文件即可享受 LittleSkin 提供的优质服务。

合并后我们会在一段时间内保留演示站的完整数据备份，如果有需要提取数据的被删除的角色或账号，可以加入 LittleSkin 官方用户交流群，与站点管理员联系。

以上的合并方案并不是最终方案，我们随时可能在不通知您的情况下对合并方案做出修改。当然，我们愿意真诚地听取您的意见。您可以点击链接或者扫描以下这个二维码参与投票并提出您的意见：https://wj.qq.com/s2/3642025/c2b6
PS：问卷设置登录后才能作答是为了防止恶意刷票。登录是通过腾讯的 OAuth 授权给腾讯问卷获取你的账号信息（而不是授权给我们），我们不会（也不能）获取你的 QQ 或 WeChat 账号的任何信息。
如果你对合并有任何问题，都可以加入 Blessing Skin 交流群 573741451 讨论。如果您想体验 LittleSkin，请访问 https://littleskin.cn。
