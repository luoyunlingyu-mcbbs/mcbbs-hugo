Tips—有用的提示
   让你的游戏内容更丰富！


这是个什么模组？
这是一个简单的辅助类模组，它添加了一个类似于基岩版和中国版的世界加载提示
它的缺点是不支持1.13-1.15的任何版本！
它只支持1.12.2和1.16的部分Minecraft版本
当然，它是支持自定义的 (有关自定义的内容将在下文中讲到)
这个模组适用于模组整合包中
原贴作者原话 ↓
This mod will display useful tips on certain game screens such as the world loading screen. The tip shown will be cycled out every five seconds. The pool of tips can be expanded by resource packs, other mods, and even modpacks.




关于汉化
如果你只是使用原版提示，你可以使用酒石酸的I18n加载汉化提示文本
如果你想自定义提示文本，那么请耐心读完本帖
有关自定义提示的语言配置会在下方讲到



下载地址
本帖暂不提供本地下载地址
如需下载请前往Curse发布贴



配置文件解析
注意，本模组的配置文件比较重要，请仔细阅读后再使用
首先它的配置文件存放在 .minecraft/config/tips.cfg 中
可配置内容如下:
# Configuration file

general {
# Determines whether or not the default tips should be possible. This includes tips added directly by other mods through their language files. [default: true]
    B:allowDefaultTips=true
#是否启用默认提示文本. [默认:开启]
B:开启/关闭默认提示文本=true/false

# A list of custom tips added by the user or modpack. [default: ]
S:customTips <
>
# 用于你整合包的自定义提示文本.
S:自定义提示文本<
>

# The color of the actual tip text. [default: FFFFFF]
S:textColor=FFFFFF
#正文的颜色. [默认:FFFFFF(即白色)]
S:文本色=FFFFFF

# The color of the top/title text for the tip. [default: FFFF55]
S:titleColor=FFFF55
# 顶端的标题颜色. [默认:FFFF55(即黄色)]
S:标题色=FFFF55

# The amount of offset the tip text should have from the left of the screen. [range: 0 ~ 2147483647, default: 5]
I:xOffset=5
# 提示文本向左方的偏移指数. [范围:0~2147483647，默认:5]
I:偏移指数=5

# The amount of offset the tip text should have from the bottom of the screen. [range: 0 ~ 2147483647, default: 40]
I:yOffset=40
# 提示文本向底部偏移指数. [范围:0~2147483647，默认:40]
I:偏移指数=40
}




自定义提示文本配置教程
在你看我写的教程之前，如需作者的原教程，可点击右方查看 戳我进入
本模组是通过扫描语言文件加载提示文本的，因此你需要装载 Resource Loader 模组 Curse发布贴
装载此模组后启动游戏，你会发现在 .minecraft 目录下多出一个 resources 文件夹
你可以在 resources 内创建一个新文件夹(名称自拟，最好不带中文)
然后再在里面创建一个名为 lang 的文件夹
再 lang文件夹 里面创建一个后缀为 lang 的文本文档文件(请根据本地语言命名，如大陆地区可命名为 zh_cn.lang)
接着，回到 .minecraft/config/tips.cfg 中
找到这一行文字
# Determines whether or not the default tips should be possible. This includes tips added directly by other mods through their language files. [default: true]
    B:allowDefaultTips=true
把 B:allowDefaultTips=true 中的 true 改为 false
接着找到这一行文字
# A list of custom tips added by the user or modpack. [default: ]
S:customTips <
>
在 S:customTips <> 的 <> 中输入一些字符串
格式：tips.自定义名称(英文).序号
比如这样：tips.modpack.1
那么在配置文件中显示的情况就会是这样:
# A list of custom tips added by the user or modpack. [default: ]
S:customTips <
tips.modpack.1
>
数量不限，具体要看你写的量
完成后，回到刚刚创建的 lang 文件中
将你要写的提示文本加上去
格式：tips.modpack.1=自定义文本
比如这样：tips.modpack.1=Minecraft
那么在配置文件中显示的情况就会是这样:
# 文本标题(可自拟，最前方要带上#)
tips.modpack.1=Minecraft
完成后就可以启动游戏了
