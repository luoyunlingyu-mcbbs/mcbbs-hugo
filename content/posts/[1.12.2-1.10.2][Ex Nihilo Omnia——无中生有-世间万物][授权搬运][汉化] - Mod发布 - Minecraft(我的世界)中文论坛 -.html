[Mod简介]

相信无中生有这个Mod大家都不陌生，很棒的一个Mod
可惜原作者弃坑，版本停留在了1.8.9
[1.8.9-1.6.4][Ex-Nihilo——无中生有]

于是Raven2396制作了这个Mod，可以理解为是E Nihilo的升级版
这个Mod有着无中生有1和2的内容，但是更接近1
更多教程请见原搬运帖

关于译名标准化

我询问过作者Omnia应该怎么翻译
作者的回复是"Everything"，应该是作为一个强调作用
于是这个Mod我决定翻译为
无中生有-世间万物



[详细介绍]

这款Mod在内容上与无中生有大同小异，那么来看看都有些什么吧

锤 Hammer
  • 资源的生产

手持锤子破坏，可以转化方块

  • 圆石→砂砾
  • 砂砾→沙子
  • 沙子→尘土
  • 沙石→沙子
  • 地狱岩→地狱砂砾 [受重力影响]
  • 末地石→末影砂砾 [受重力影响]

可能是由于正在开发的原因，我还没成功转化过地狱岩和末地石

  • 矿物的处理


将砂砾放入筛子中，可能会得到任意的破碎矿石
将4个破碎矿石合成，能够得到矿砂
用锤子砸矿砂可以得到4个粉碎矿，4个粉碎矿可以合成矿沙
用锤子砸矿沙可以得到5个矿粉，4个矿粉可以合成矿粉尘
矿砂、矿沙、矿粉尘都可以直接烧成锭，不过还是烧矿粉尘划算一点


  • 锤的制作
木板可以换成其它种类的


当然，圆石、铁锭、金锭、钻石也是可以的



钩子 Crooks
用钩子右键生物可以将它拉到你的身边，左键可以击退它们，不过没有伤害
用钩子挖掘树叶会增加一倍掉落树苗和苹果的几率
除此之外，还能用骨头合成骨钩



蚕 Silkworms
用钩子挖掘树叶，有很小的几率掉落蚕


手持蚕，对树叶右键，树叶会变白，并且开始蔓延


足够的时间之后，树叶会全部变白


挖掘白色的树叶会掉落线，树叶越白，掉的越多，但是不会掉落树苗
在挖掘树叶前，请让蚕儿们完成他们的工作

由于蚕儿的营养是牛肉的六倍，为什么不试试直接把它们烤了吃呢？



物品 Items
  • 灰烬
烧制一根木棍就能得到


灰烬可以像骨粉一样催熟作物和植物
也可以和骨粉、粘土合成瓷土，瓷土是相当有用的


  • 瓷桶
三个瓷土可以合成一个未烧制的瓷桶


把它烧一烧就能用了，用途跟铁桶一样


  • 小石头
如果用锤子锤石头，你就能够获得小石头
你可以试试你能把它们扔出多远


也可以用来合成圆石


在配置文件中可以修改合成圆石所需的小石子数量，范围1~9，如果设置为0那么就是关闭合成

  • 孢子
这是个有趣的东西，右键泥土，可以让它变成菌丝，似乎它还和牛有特殊的反应
[手持孢子右键牛可以将牛变成蘑菇牛]

坩埚 Crucible
足够的瓷土可以做成未烧制的瓷坩埚


当然还是得烧一烧才能用


它可以用于转化岩浆

  • 熔化
将坩埚放在热源上


现在尝试添加点什么进去


现在只要等待就好了，热源温度越高，熔化速度越快



筛子 Sieve
筛子可以用任何类型的木板制作


然后在里面放上筛网
现在有两种筛网


如果你不想这么麻烦，你可以修改配置文件，让筛子在放置时自带筛网，而且拥有无限的耐久


手持泥土右键筛网放进去


然后不断右键它


接下来你就能得到一些东西


如果你的运气够好，你可以得到一些好东西
具体有什么呢？

泥土：小石子、小麦种子、西瓜种子、南瓜种子、甜菜种子、胡萝卜种子、马铃薯种子、甘蔗种子、各种树的种子

砂砾：燧石、煤炭、青金石、钻石、破碎矿石
沙子：可可豆、仙人掌种子、孢子、粉碎矿石
灵魂沙：石英、地狱疣、恶魂之泪
地狱砂砾：地狱破碎矿石

筛机 Sifter
当你不断驱动筛子来回运作时，我想你都要疯了
当然，我们可以更快♂一点，看看这是什么，筛机！
使用不同材质的筛机，速度是不一样的
你还可以把它带在身上


桶 Barrels
事实证明桶可以被各种材料制成


桶是非常有用的，你可以像这样制作它们



  • 制作泥土
你可以把植物塞进这个桶里，塞满它




然后等一会


最后你会得到一块泥土


  • 收集雨水
桶可以在下雨天时用来收集雨水，就像这样


  • 粘土
桶里装满雨水后你可以放入尘土，最后你会得到一块粘土

  • 地狱岩和末地石
如果你的桶可以放入岩浆且不会被烧毁，你可以拿着红石或者萤石右键，你会得到一块地狱岩或者末地石‘

  • 巫水
如果你足够幸运的从筛子里获得了孢子，你可以拿着它右键水桶，等待一会后，水会变成巫水
如果你接触巫水，你会获得一些Debuff，爬行者会变成高压爬行者，骷髅会变成凋灵骷髅（岂不美哉？）
将巫水盛放在桶里，拿着沙子右键，可以获得灵魂沙

  • 召唤史莱姆
往装满水的桶里倒入牛奶


水的颜色就会变绿


等待一会，就会出现史莱姆


  • 玩偶
可以用钻石或者绿宝石和瓷土合成


  • 召唤烈焰人
合成一个愤怒玩偶，右键装满岩浆的桶，然后等待...


  • 召唤末影人
合成一个幽灵玩偶，右键装满巫水的桶，然后等待...



配置 Config
Pretty much everything about the mod can be changed, there is a lot to cover so if you are interested, I recommend going through the file.     You can use JSON to create your own recipes for compost, hammering, sifting, melting, and add heat sources for the crucible. You can even disable all the built in ones to start from scratch you if you so desire. (IMPORTANT: each folder contains an "example.json". This example file is NEVER LOADED! It is purely for reference!)     The mod will try to load in as many ores that it can find, but sometimes it has trouble. If you notice that your modpack has, for example, Nickel, but Ex Nihilo isn't adding it, go to the config and change B:"force nickle" to true. If nothing is loading for whatever reason, you can have it load in all ores it knows about.  World gen Ex Nihilo Omnia offers a config option to have the world generate as the original skyblock 2.1 map, a more hardcore version in which you spawn on a floating piece of dirt with a tree, and a middleground version of skyblock 2.1 with no grass. You can chose which biome the entire world will generate as, or keep it with all of them. All of these options can be found in the "world generation" section of the config.

兼容性 Compatibility
The mod should find most ores I've told it about, but if it doesn't, you can help it out in the config like I said above.  All ores that it adds can be set to be oredict compatible in the config.  Crooks and Hammers are veinminer ready!  Barrels, sieves, crucibles, and infected leaves all work with with WAILA.  All ores added will work in the Tinkers' smeltery  I will start working on JEI recipe menu things for sifting, hammering, composting, etc soon.
