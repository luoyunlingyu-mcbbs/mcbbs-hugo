EasyVar基本介绍
简单设置变量并根据变量的值执行命令,高度自定义且简单的变量插件(5行命令实现在线奖励发放)

开源
  1. https://github.com/abmcar/EasyVar
复制代码


前言
    曾经在开服务器的时候需要一个插件来记录玩家的自定义变量，然而当时的我翻遍了bbs也没有找到一个能用的插件。多年之后的一天，打完多校头昏眼花的我偶然想到了这件事情，恰逢暑假集训刷题无聊，产生了自己写一个插件的的想法，最终经过两天的愉快摸鱼，终于完成了该插件的最初版本。
    由于第一次写插件且之前没有开发经验，此插件源码急需dalao纠正且不建议任何初学者学习。


功能
  • 自定义全局/玩家变量
  • 增加修改记录变量
  • 根据变量的值以OP/普通权限执行命令
  • 每隔一定时间自动执行脚本


PAPI
    感谢呆瓜小黑的pr让EasyVar支持papi    %ev_自定义变量% 即是Ev变量


我能拿它做什么？
  • 设置一个一次性脚本对初始化玩家的进服权限和物品
  • 设置一个全局变量来统计全服玩家的行为次数
  • 设置一个针对玩家变量的脚本给予玩家某个行为的累计奖励
  • 设置一个脚本发放玩家的登陆累计奖励
  • 设置一个玩家变量记录玩家的某个贡献
  • 设置一个脚本对某些变量进行更改
  • 记录玩家的在线时长并发放奖励
  • 等等......

举个栗子

config.yml

  1. runTask: true
  2. runTaskTime: 0.5
  3. runPlayerScriptName: basePlayerScript
复制代码

basePlayerScript.yml

  1. varType: global
  2. varName: baseVar
  3. requireValue: 0
  4. isOp: false
  5. commands:
  6. - evdisplay script run onlineReward
  7. - evdisplay add player <player> onlineTime 1
复制代码
onlineReward.yml

  1. varType: <player>
  2. varName: onlineTime
  3. requireValue: 3600
  4. isOp: true
  5. commands:
  6. - evdisplay reduce player <player> onlineTime 3600
  7. - give <player> minecraft:diamond 1
复制代码
简单实现了在线半个小时奖励玩家一颗钻石的功能
命令如下：
  1. ev script set onlineReward <player> onlineTime 3600 true
  2. ev script add onlineReward give_<player>_minecraft:diamond_1
  3. ev script add onlineReward evdisplay_reduce_player_<player>_onlineTime_3600
  4. ev script add basePlayerScript evdisplay_script_run_onlineReward
  5. ev script add basePlayerScript evdisplay_add_player_<player>_onlineTime_1

关于开发使用：

直接调用EasyVar.getVarManager()

主要方法如下：
  1. Integer getPlayerValue(String playerName, String varName);
  2. Integer getGlobalValue(String varName);
  3. void setGlobalValue(String varName, Integer value);
  4. void setPlayerValue(String playerName, String varName, Integer value);
复制代码


例子：
  1. Integer tempInteger = EasyVar.getVarManager().getPlayerValue("abmcar","test");
复制代码







权限
  1. ev.var.show        //查看变量
  2. ev.var.set           //修改变量
  3. ev.script.show     //查看脚本命令
  4. ev.script.set        //修改脚本命令
复制代码



命令
  1. 使用/ev show player <playerName>查看该玩家拥有变量,playerName为玩家名字
  2. 使用/ev global来查看全局变量
  3. 使用/ev set player <playerName> <varName> <varValue>修改玩家变量
  4. 使用/ev set global <varName> <varValue>修改全局变量
  5. 使用/ev add player <playerName> <varName> <varValue>增加玩家变量
  6. 使用/ev add global <varName> <varValue>增加全局变量
  7. 使用/ev reduce player <playerName> <varName> <varValue>减少玩家变量
  8. 使用/ev reduce global <varName> <varValue>减少全局变量
  9. 使用/ev script set <scriptName> <varType> <varName> <varValue> <isOp> 创建或设置脚本
  10. 例如/ev script set script1 global test 10 true 创建script1当全局变量test大于等于10时以OP执行命令
  11. 例如/ev script set script2 Abmcar test1 10 false 创建script2当玩家Abmcar的变量test1大于等于10时以自身权限执行命令
  12. 使用/ev script add <scriptName> <command>添加脚本执行命令
  13. 使用/ev script delete <scriptName> <command>删除脚本执行命令
  14. 命令中的空格请用_代替,例如/ev script delete testScript ev_show_global
  15. 使用/ev script show <scriptName> 查看脚本执行命令
  16. 使用/ev script run <scriptName> 执行脚本
