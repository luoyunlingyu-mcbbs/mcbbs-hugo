关于
The New Economy是一个功能齐全的模块经济插件，用于Bukkit服务器，支持虚拟货币和基于项目的货币。它也支持与基于项目的货币的离线玩家进行交易。我们还支持多个世界和多种货币.

指令


/tne backup - 创建所有服务器数据的备份.

/tne balance <玩家> [世界] [货币]  - 检索指定世界玩家的余额.

/tne build - 显示当前正在运行的TNE版本.

/tne caveats - 显示此版本TNE的所有已知警告.

/tne create <玩家> [余额]  - 创建一个新的经济账户.

/tne debug - 切换控制台调试模式.

/tne delete <玩家>  - 删除玩家账户.

/tne extract -使用附加的用户名提取所有玩家余额，以进行与数据库相关的调试.

/tne id <玩家> -检索玩家的TNE UUID.

/tne maintenance - 切换经济维护模式，禁用所有经济功能.

/tne menu <玩家> - 指定玩家打开菜单（菜单英文）.

/tne purge - 删除所有具有默认余额的玩家帐户.

/tne recreate -尝试重新创建数据库表。 警告: 这将删除数据库中的所有数据.

/tne reload <文件> - 保存通过命令进行的修改，并重新加载配置文件.

/tne reset -从数据库中删除所有与经济相关的数据.

/tne restore - /tne extract之后恢复位于extracted.yml中的所有余额.

/tne save -强制保存所有数据.

/tne status <玩家> [状态] - 显示或设置帐户的当前帐户状态.

/tne upload -将TNE调试和最新服务器日志上载到pastebin.com，并提供每个服务器日志的链接.

/tne version -显示当前正在运行的TNE版本.

/currency rename <货币> <新名字> -将货币重命名为其他名称.

/currency list [世界] - 显示世界可用的货币.

/currency tiers [货币] [世界] - 显示货币的等级.

/language current - 显示您为帐户设置的当前语言.

/language list - 列出此服务器上的可用语言.

/language reload - 重新加载所有语言文件.

/language set <名字> - 将当前语言设置为指定的语言.

/tnem info <模块> - 显示有关模块的一些信息.

/tnem list -列出所有已加载的TNE模块.

/tnem load <模块> - 从模块目录加载模块.

/tnem reload <模块> - 从模块目录重新加载模块.

/tnem unload <模块> - 从服务器卸载模块.

/money balance [世界] [货币] - 显示您当前的余额.

/money convert <量> <to 货币[:world]> [来自货币[: 世界]]  - 将您的部分货币转换为另一种货币.

/money give <玩家> <量> [世界] [货币] - 为您的经济增加资金，并将其交给玩家.

/money note <量> [货币] - 出于存储/交易目的，使您的虚拟货币变为实际货币.

/money pay <玩家> <量> [货币] - 使用您的一些货币来支付其他玩家.

/money set <玩家> <量> [世界] [货币] - 设置玩家的货币.

/money take <玩家> <量> [世界] [货币] - 从您的经济中移除资金，特别是从玩家的余额中移除.

/money top [页] [货币: 名称] [世界: 世界] [limit:#] - 具有最高余额的玩家列表.

/transaction away [页 #] - 显示自您上次上次以来错过的交易.

/transaction history [玩家: 名字] [页:#] [世界:名字/全部] [玩家:名字] - 查看交易历史的详细细分.

/transaction info <uuid/id> - 显示有关交易的信息.

/transaction void <uuid/id> - 撤消先前完成的交易.
