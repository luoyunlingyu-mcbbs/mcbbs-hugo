🔎 开发
  • 位于config/avaritia/singularities文件夹下，可以利用json自定义奇点：

  1. {
  2.   "name": "singularity.avaritia.bronze", // 在语言文件中本地化的名称。
  3.   "colors": [
  4.     "d99f43",   //覆盖色。
  5.     "bb6b3b"  //背景色。
  6.   ],
  7.   "materialCount": 1000, //默认是1000个合成一个对应奇点。
  8.   "timeRequired": 240,  //在中子态素压缩机中产生对应奇点所需tick。
  9.   "conditions": [    //启用条件。
  10.     {
  11.       "type": "forge:not",
  12.       "value": {
  13.         "tag": "forge:ingots/bronze",
  14.         "type": "forge:tag_empty"
  15.       }
  16.     }
  17.   ],
  18.   "ingredient": {  //输入中子态素压缩机的物品或tag。
  19.     "tag": "forge:ingots/bronze"
  20.   },
  21.   "enable": true //是否启用。
  22. }
复制代码


  • 利用crafttweaker添加配方：

  1. mods.avaritia.CompressionCrafting.addRecipe("name",input, inputCount, timeRequierd);//添加中子态素压缩配方。
  2. mods.avaritia.CompressionCrafting.remove(output);//移除中子态素压缩配方。
  3. mods.avaritia.ExtremeTableCrafting.addShaped("name",output, ingredients);//添加无尽工作台有序配方。
  4. mods.avaritia.ExtremeTableCrafting.addShapeless("name",output, ingredients);//添加无尽工作台无序配方。
  5. mods.avaritia.ExtremeTableCrafting.remove(output);//删除无尽工作台配方。
复制代码


  • 更改无尽催化剂的配方：

由于自定义奇点的存在，无尽催化剂的配方是根据加载的奇点动态变化的，你可以自定义添加除奇点以外的物品，且此配方类型只能产出无尽催化剂，更改result无法更改产品。

  1. {
  2.   "type": "avaritia:infinity_catalyst_craft",//无尽催化剂的配方类型。
  3.   "category": "misc",
  4.   "ingredients": [
  5.     {
  6.       "item": "minecraft:emerald_block"
  7.     },
  8.     {
  9.       "item": "avaritia:crystal_matrix_ingot"
  10.     },
  11.     {
  12.       "item": "avaritia:neutron_ingot"
  13.     },
  14.     {
  15.       "item": "avaritia:cosmic_meatballs"
  16.     },
  17.     {
  18.       "item": "avaritia:ultimate_stew"
  19.     },
  20.     {
  21.       "item": "avaritia:endest_pearl"
  22.     },
  23.     {
  24.       "item": "avaritia:record_fragment"
  25.     }
  26.   ],
  27.   "result": {
  28.     "item": "avaritia:infinity_catalyst"//固定的产品。
  29.   }
  30. }
