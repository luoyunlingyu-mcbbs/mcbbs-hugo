i  前言
注意！该模组需要以下模组作为前置：TerraBlender ；如果是 Fabric 版本还需安装 Fabric API 和 Forge Config API Port
本帖图片较多，且多数图片均使用 SMMS 图床上传。如果您是第一次打开本帖并遇到图片未完整加载出来，请确保网络通畅并耐心等待即可
有没有想过，在 Minecraft 里面尽情探索现实世界的自然群系会是一番什么样的感受？这个模组将会告诉你答案！本模组为 MC 1.16.5+ 而生，以多种方式将原版游戏的世界改头换面，将一个更加真实的自然世界呈现给你。本模组加入了多种全新的生物群系以及有关的方块，物品等，其中有很多生物群系都参考了现实世界里的自然群系（当然也有一些生物群系是架空虚构的）。这个模组让原版游戏的世界变得更加丰富多彩，也极大的扩展了原版游戏的探险玩法

此外，这个模组也能让你对现实世界的自然有着更多的了解，让玩家们知道，原来我们生活的这个世界也有着许多各式各样，风格独特的自然群系。你知道约书亚树，巨人柱，红火球帝王花是什么样子的吗？你知道北方森林，海岸红杉林，猴面包树草原是怎样的景观吗？这个模组便会带你尽情体验和感受它们，哪怕你没有到过这些地方，也没有见识过它们，也能获得极佳的身临其境的感觉

对于那些热爱探索的玩家来说，这个模组无疑是个绝佳的选择，多达70余种的生物群系完全能满足探险爱好者们对更多奇妙地域探索的好奇心。而对于那些热爱建筑的玩家来说，这个模组的多种木制方块，尤其是染色木板也能给他们的建筑设计提供更多新的思路，建造更多使用不同木材的建筑也是一种不错的选择

《未至之域》现有附属模组力（大喜）！现已推出非官方附属模组：Areas Undefined（暂译为“未定之地”）。该附属模组专注于对原版游戏的生物群系进行改进，使其与《未至之域》的生物群系更加协调。截至目前，Areas Undefined 对 30 个常见的原版主世界的生物群系进行了改进，未来也可能支持其他模组的生物群系。点击上方图片即可了解更多，以及下载！


i  简介
这是一个旨在让原版游戏的生物群系变得更加缤纷多彩的模组。该模组对原版的地形和生物群系的生成进行的较大幅度的改进，让原版游戏的主世界焕然一新。本模组加入了多种基于现实自然的生物群系，多种树木和其他植物，多种自然和装饰性方块和其他一些装备和物品等。正如这个模组的名称所描述的那样，“Regions Unexplored”，即这些生物群系（地域）此前从未有过人到达和探索过，以此体现这些生物群系的原生态的特点，也相信这些生物群系一定会给各位探险者带来许多来自原生自然的惊喜


该模组现阶段为原版游戏带来了多达70余种现阶段共计72种全新的生物群系。很多生物群系是基于现实世界中的自然群系来设计的，从原始竹林到北美海岸红杉林，从内陆草原到河口三角洲，从巨人柱沙漠到寒带苔原。上至高山，下至洞穴，不同的生物群系具有着其独特的风格和特有的元素。相信你一定能在本模组内体验到探索真实世界的生物群系的乐趣。此外，模组也加入了一些架空虚构的生物群系，依旧充满了天马行空的想象和极具特色风格，例如遍布着红石晶体或五颜六色的水晶簇的洞穴，以及能让你感受到远古 MC 版本的诸多元素的“alpha”森林。以下图例仅仅是展示了本模组中的一小部分的生物群系。实际上，未在下图展示的其他的生物群系依然精彩十足，完全值得广大玩家们的尽情探索








本模组也加入了多种各式各样，五光十色的的方块，包括各种自然方块，树木方块以及对应的木制方块等。你能在不同的生物群系找到特有的方块。同时，不少的方块也具有装饰价值，利用这些方块，你也能建造出许多独具风格，令人眼前一亮的建筑


各种各样的自然方块。从草地和泥土，苔石到矿石，你总能找到在不同的生物群系里面找到独特的自然方块


各种各样的的树木方块，包括多种原木和树叶。不同的树木方块都有着不同的纹理和色彩


以及各种各样的木制方块，涵盖由各种原木加工而成的木板等，以及基于原版木板经染色而制成的染色木板

本模组的乐趣还远不至此。多种五彩缤纷的花卉，形态各异的植被也值得玩家的寻找和采集。挖掘模组中的各种矿石，并用它们来制作一些武器和装备也是一种不错的选择（这是未来计划的一部分）。该模组仍在不断扩充和完善中，未来还会加入更多精彩的生物群系，更多的植物和更多的玩法等，甚至未来的更新还将不仅限于原版的主世界（是的，作者已经有了改进下界的打算，并放出了最新测试版）。想体验到更多精彩内容？就请大家敬请期待罢！
