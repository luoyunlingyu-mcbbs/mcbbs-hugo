RankFight

一个简单但有用的单挑插件





简介：
RankFight 是一个简单但有用的单挑插件，它允许玩家在多个竞技场中指定其中一个开始单挑，获胜者将可以得到积分，积分可在商店中兑换物品，同时，借助 PlaceholderAPI，您还可以向玩家展示玩家当前积分和全服排名。配置可自定义。

配置文件：
  1. # 排位积分间隔
  2. CreditInterval: 100
  3. # 胜利获得积分数
  4. CreditWin: 10
  5. # 游戏队列最大等待时间，单位秒，设置为 -1 则无限制
  6. QueueTimeoutSec: 30
复制代码


指令：

  2. "/rankfight join [竞技场名称（可选）]-- 加入或创建一场单挑",
  3. "/rankfight list -- 查看可用的竞技场列表",
  4. "/rankfight status -- 查看自己的排名和积分",
  5. "/rankfight shop -- 查看积分商店",
  6. "/rankfight admin status <玩家名> -- 查看指定玩家的排名和积分",
  7. "/rankfight admin addArena <竞技场名称> <x1> <y1> <z1> <x2> <y2> <z2> -- 创建一个竞技场",
  8. "/rankfight admin removeArena <竞技场名称> -- 移除一个竞技场",
  9. "/rankfight admin addItemGift <所需积分> -- 添加手持物品为商店奖励",
  10. "/rankfight admin removeItemGift -- 移除手持物品为商店奖励",
  11. "/rankfight admin setCredit <玩家名> <积分> [商店积分（可选）] -- 设置指定玩家的积分和商店积分",
  12. "/rankfight admin reload -- 重载插件配置文件并强制存储数据到磁盘",
复制代码


权限：
  1. rankfight.admin -- 使用管理员指令
复制代码


变量（需要安装 PlaceholderAPI 2.10.10 或以上版本）：

  2. %rankfight_rank% -- 全服排名
  3. %rankfight_credit% -- 积分数
  4. %rankfight_shopCredit% -- 可用的商店积分数
