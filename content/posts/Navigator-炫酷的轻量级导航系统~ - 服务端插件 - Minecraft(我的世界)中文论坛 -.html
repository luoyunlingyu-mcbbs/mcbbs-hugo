配置文件：
  1. sql:
  2.   user: #用户，要有对应权限
  3.   passwd:  #密码
  4.   url: localhost:3306
  5.   database: #必须是已经创建的数据库
复制代码
插件第一次被启用时会生成文件并默认没有被启用，需要填写正确的sql以后再重启服务器。
第二个文件不需要自己填写，在用/navigator edit 名字 时会自动生成。
cmds:[]表示一串点踩完时执行的命令
  1. cmds: []
复制代码
命令：/navigator set 名字 导航点（给某个玩家设置任务）
         /navigator edit 导航点（新建导航点）
         /navigator reload （重载配置文件）
         /navigator remove 导航点 索引（移除这个导航点）
权限就不写啦，用luckperms的自动补全！
注：请用remove安全删除导航点，不要手动删配置文件，否则可能出现indexOutOfBound

第一次更新：
对不起对不起，之前的插件使用placeholder的时候有bug，现已修补
placeholder:
%nav_name%下一个任务名
%nav_direction%用箭头表示东南西北
%nav_distance%离导航点的距离

第二次更新：cmds（做完任务执行的命令）支持placeholder啦！
e. g.:kick %player_name%
